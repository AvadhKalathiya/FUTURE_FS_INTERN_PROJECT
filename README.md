## 🔹 Task 2 — Client Lead Management System (Mini CRM)

### 📖 Description

A lightweight **Customer Relationship Management (CRM) system** built to manage and track client leads generated from website contact forms.

This application allows administrators to organize leads, track communication status, and manage follow-ups efficiently.

### ✨ Features

* Lead management dashboard
* Lead information tracking (Name, Email, Phone, Source)
* Lead status updates (New / Contacted / Converted)
* Notes and follow-up tracking
* Backend REST API integration
* Database storage with MongoDB
* Secure authentication system

### 🛠 Tech Stack

* Next.js / React
* Node.js
* Express.js
* MongoDB Atlas
* Tailwind CSS

### 🎯 Skills Gained

* CRUD operations
* RESTful API development
* Database design and management
* Authentication & session handling
* Backend-frontend integration
* Business workflow automation

# 🚀 Running Projects Locally

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/AvadhKalathiya/FUTURE_FS_INTERN_PROJECT.git
```

---

### 2️⃣ Navigate to the Project Folder

```bash
cd FUTURE_FS_INTERN_PROJECT
```

---

### 3️⃣ Install Dependencies

```bash
npm install
```

---

### 4️⃣ Run Development Server

For React / Next.js projects:

```bash
npm run dev
```

or

```bash
npm start
```
