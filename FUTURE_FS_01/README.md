## 🚀 Getting Started (Local Setup)

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/AvadhKalathiya/your-repo-name.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd your-repo-name
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run Development Server

```bash
npm run dev
```

Now open:

```
http://localhost:3000
```
